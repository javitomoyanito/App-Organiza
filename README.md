CameraApp - Proyecto Android (Kotlin)
------------------------------------

Este proyecto abre la cámara, permite tomar una foto, muestra la foto en pantalla y la guarda en la galería.

Cómo usar:
1. Importa o abre este proyecto en Android Studio.
2. Conecta un dispositivo físico (recomendado) o usa emulador con cámara.
3. Ejecuta la app.
4. Permite los permisos cuando Android los solicite.
5. Toma una foto; se mostrará en pantalla y se guardará en la galería (DCIM/Camera).

Notas:
- Min SDK 24. Si usas Android 10+ (API 29+) se usa MediaStore para guardar en la galería.
- Si te faltan recursos como launchers, Android Studio te pedirá generarlos automáticamente.
